About Project

The Indonesian Merupakan Web yang akan diproseskan bagaimana caranya untuk melakukan dalam pembuatan proyek tersebut, dengan ini saya melakukan pembuatan dengan video tersebut sebelumnya itu, kalian harus menguasai HTML,CSS dan JS, supaya bisa menuju bootstrap project tersebut untuk pembuatan video, digunakan sebuah program davinci resovlve atau premiere, yang bertujuan mempresentasikan tutorial dalam web tersebut, maka web ini kemungkinan dijelaskan secara pengertian dalam koding tersebut.

Untuk pembuatan proyek web, perlunya template Bootstrap, HTML, CSS dan JS, definisi dari The Indonesian merupakan web yang berisi informasi yang mengenai budaya, makanan dan wisata dari indonesia, jadi tujuannya supaya orang bisa melihat informasi dengan singkat dan jelas, bahkan web masih versi pertama belum sampai versi terakhir karena versi diperlukan kreatifitas supaya web menjadi bagus dan informatif, maka untuk video perlunya ada setelah halaman ke-2 atau setelah proses, yang bertujuan jika kalau masih menunjukan hal yang dalam dan modifikasi terhadap web tersebut.

Untuk membuat ini perlu namanya Bootstrap yang dimana sebuah framework yang diciptakan oleh Mark Otto dan Jacob Thornton pada tahun 2011, maka bootstrap menjadikan programmer mudah dipakai dibanding css biasa, maka itu proses tersebut dari point ke point yaitu, jumbotron dan navbar yang digabung jadi sebuah header yang berbeda, lalu ada informasi apalagi yang nanti bahas divideo adalah cara transisi lewat javascript supaya aplikasi dinamis. JavaScript

JavaScript atau kita singkat menjadi JS merupakan bahasa pemrograman jenis interpreter, sehingga kamu tidak memerlukan compiler untuk menjalankannya. JavaScript memiliki fitur-fitur seperti berorientasi objek, client-side, high-level programming, dan loosely typed. Berikut Codingan bisa berjalan tersebut 

                  window.addEventListener('scroll', reveal);
function reveal() {
  var reveals = document.querySelectorAll('.reveal');
  for (var i = 0; i < reveals.length; i++) {
    var windowheight = window.innerHeight;
    var revealtop = reveals[i].getBoundingClientRect().top;
    var revealpoint = 150;
    if (revealtop < windowheight - revealpoint) {
      reveals[i].classList.add('active');
      } else {
          reveals[i].classList.remove('active');
      }
    }
  }
